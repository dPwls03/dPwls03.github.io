## Chapter 01. 웹 프로그래밍과 HTML5개요

<br>
> [학습내용]

웹은 WWW(World Wide Web) 즉, 세상의 전체 컴퓨터를 연결하는 거미줄을 줄여 웹이라고 부른다.

웹과 인터넷은 기본적으로 다르다.

인터넷은 인터넷 주소를 가지고 상대에게 데이터를 전송할 수 있도록 만든 통신망이다.

웹은 이 통신망 위에서 이루어지는 서비스 중 하나로, 웹 서버에 다수의 클라이언트 컴퓨터가 연결하여 상호 정보를 교루하는 시스템이다.

웹 페이지의 구조와 내용읜 HTML5 태그로, 출력되는 모양은 CSS3 스타일 시트로 작성하고 사용자 입력을 처리하거나 웹 애플리케이션을 작성하는 것은 자바스크립트로 나누어 만들어야 한다.

## Chapter 02. HTML5 기본 문서 만들기
<br>
> [학습내용]

HTML은 태그로 문서의 각 요소를 표현하여 웹 문서를 작성하는 언어이다.

### [HTML 태그들]

&lt;title&gt; : HTML 페이지의 제목으로, 타이틀 바에 출력된다.

&lt;h1&gt; ~ &lt;h6&gt; : 6개의 태그를 제공하여 문단에 제목을 붙일 수 있다.

&lt;p&gt; : 문단은 여러 단락으로 나눌 수 있으며 하나의 단락은 &lt;p&gt; 태그로 표현한다.

&lt;hr&gt; : 문단 내에서 내용의 전환이 필요한 곳에 수평선을 삽입하여 표현한다.

&lt;br&gt; : 새로운 줄로 넘기고자 할때 사용한다.

&lt;pre&gt; : 개발자가 입력한 포맷 그대로 출력할 때 사용한다.

&lt;div&gt; :  여러 HTML 태그들을 블록으로 묶는 컨테이너로 이용된다.

&lt;span&gt; : 텍스트 일부분에 특별한 모양을 주거나, 자바스크립트 코드로 텍스트 일부분을 제어하고자 할 때 사용한다.

&lt;base&gt; : 웹 페이지들이 담겨 있느 기본 URL과 웹 페이지가 출력될 윈도우를 지정하기 위해 사용된다.

&lt;meta&gt; : 웹페이지의 저작자, 문자 인코딩 방식, 문서 내용 등 다양한 메타데이터를 표현하기 위해 사용된다.

&lt;img&gt; : HTML 문서에 이미지를 삽입할 수 있다.

&lt;ol&gt;, &lt;ul&gt;, &lt;dl&gt; : 리스트 데이터를 나열할 때 사용된다.

&lt;dl&gt;, &lt;dt&gt;, &lt;dd&gt; : 정의 리스트 전체는 &lt;dl&gt;, 각 아이템에서 용어는 &lt;dt&gt;, 설명은 &lt;dd&gt;로 표현한다.

&lt;table&gt; : 표를 작성할 때 사용한다.

&lt;a&gt; : 하이퍼링크를 만들 때 사용하며, href 속성으로 만든다.

&lt;iframe&gt; : 현재 HTML 페이지 내에 내장 윈도우를 만들고 다른 HTML 페이지를 출력할 수 있다.

&lt;video&gt; : 비디오를 삽입할 때 사용된다.

&lt;audio&gt; : 오디오를 삽입할 때 사용된다.

## Chapter 03. HTML5 문서 구조화와 웹 폼

<br>
> [학습내용]

HTML 페이지 안을 검색해야 하는 검색 엔진에게 문서의 구조를 명확히 전달하는 웹페이지를 만들어야, 검색이 잘 되고 웹 사이트의 가치가 올라가기 때문에 문서의 구조화가 중요하다.

문서의 구조화를 위한 시맨틱 태그는 &lt;header&gt;, &lt;section&gt;, &lt;article&gt;, &lt;nav&gt;, &lt;aside&gt;, &lt;footer&gt;, &lt;main&gt;, &lt;summary&gt;, &lt;mark&gt; 등의 태그가 있다.

폼의 본래 목적은 웹 페이지에서 사용자 입력을 받아 서버로 전달하는데 있었다. 하지만 현재 많은 웹 애플리케이션들은 사용자 인터페이스로 사용한다.

폼 요소는 텍스트 입력 창, 버튼, 체크박스, 라디오버튼, 콤보박스 등 다양하다.

&lt;label&gt; 태그는 폼 요소와 갭션을 한데 묶어 하나의 단위로 다루어지게 한다.

## Chapter 04. CSS3로 웹 페이지 꾸미기

<br>
> [학습내용]

CSS3는 HTML 문서의 모양을 꾸미는 언어이며, CSS3로 작성한 코드르 CSS3 스타일 시트라고 부른다.

CSS3 스타일 시트를 만들 때 꼭 있어야 하는 것이 셀렉터이다.

셀렉터는 CSS3 스타일 시트를 적용할 태그를 지정하는 기증으로 여러가지가 있는데,

태그이름 셀렉터, id 셀렉터, class 셀렉터, 자식 셀렉터, 자손 셀렉터, 전체 셀렉터, 속성 셀렉터, 가상 클래스 셀렉터 등 다양하다.

박스 모델이란 HTML 태그가 콘텐츠, 패딩, 테두리, 여백으로 구성된 박스라는 개념을 뜻한다.
